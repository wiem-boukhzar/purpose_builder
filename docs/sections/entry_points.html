Entry Points & Runtime Options

Compare the four Streamlit entrypoints, their LLM connectivity, and how to switch between SHACL and child-map validation at runtime.

Entry Point Matrix

File Primary use LLM endpoint Validation
frontend/app.py Internal deployment Apollo (OpenAI-compatible) SHACL (pySHACL)

Environment Variables

  • APOLLO_CLIENT_ID, APOLLO_CLIENT_SECRET (and optional APOLLO_TOKEN_URL, APOLLO_BASE_URL, APOLLO_MODEL) — for Apollo deployments.
  • PURPOSE_LOG_PATH (optional) — override default JSON log location.

Final Validation Flow

  • Interactive disease checks use the MONDO child map to keep the chat responsive.
  • When the user clicks “Run Final MONDO Validation”, the app runs pySHACL before enabling the JSON download.
