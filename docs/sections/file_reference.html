File Reference Cheat Sheet

Quick lookup for the modules, scripts, and assets that power the Purpose Creator stack.

Streamlit Entry Points

Path Purpose
frontend/app.py Hosted Streamlit app (SHACL validation, Capgemini LLM endpoint).
frontend/app.py Local Streamlit app wired to Apollo backend; useful in air-gapped workshops.

Purpose Application Package

Path Highlights
purpose_app/common.py LangGraph assembly, reply generation, session state helpers.
purpose_app/logging_utils.py JSON logger; central hook for analytics and monitoring.
purpose_app/__init__.py Exports build_graph, state models, and convenience functions.

Ontology Validation Package

Path Highlights
ontology_validation/ontology_loader.py Loads DO graph, builds label index and child map, exposes singletons.
ontology_validation/disease_linker.py RapidFuzz scoring, LinkedDisease / UnknownDisease models, label resolution.
ontology_validation/validator.py SHACL validation entry point returning ValidationReport.
ontology_validation/suggestions.py SPARQL queries for direct subclasses (refinement options).
ontology_validation/shapes.ttl Two SHACL shapes: minimum disease count and generic concept block.

Documentation & Assets

Path Purpose
docs/DETAILED_OVERVIEW.html Primary documentation hub with deep-dive subpages.
docs/sections/*.html Topic-specific deep dives (LangGraph, ontology, testing, ops).
docs/SHACL_vs_Precomputed.md Textual comparison of SHACL vs. child-map for policy discussions.
data/ontology/doid.owl Pinned Disease Ontology export (used at runtime and in tests).

Tests

Path Coverage
tests/test_linker.py Fuzzy matching, abbreviations, tie-breaking.
tests/test_validator.py Generic vs. specific disease validation logic.
tests/test_suggestions.py Child suggestion accuracy for umbrella diseases.
tests/test_smoke.py End-to-end conversation sanity checks (multi-disease, typos).
