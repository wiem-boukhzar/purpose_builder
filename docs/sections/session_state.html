Session State Reference

Every user interaction updates Streamlit's session_state. Understanding these keys is essential for extending the agent.

Core Conversation Keys

Key Description Example
messages Ordered list of chat messages. Each item contains the canonical content and optional display (e.g., “option 2 (type 2 diabetes mellitus)”). {"role": "user", "content": "type 1 diabetes mellitus", "display": "option 4 (type 1 diabetes mellitus)"}
structured_data Normalised purpose fields (diseases, TA, medication, collaborator, product_development). {"diseases": ["type 1 diabetes mellitus"], "TA": "Metabolic Disorders"}
thinking Boolean spinner flag while awaiting an LLM response. True immediately after the user submits a message.

Validation Keys

Key Description When updated
ontology_linked_diseases List of dictionaries containing id (DOID), iri, label, score, and source_text. After every successful call to link_many().
ontology_generic_diseases Generic parents needing refinement, including child suggestions ({"label": …, "suggestions": [{"iri": …}]}). After SHACL or child-map validation detects subclasses.
ontology_unknown_diseases Unknown terms with fuzzy suggestions for the assistant to surface. Whenever a disease string falls below the RapidFuzz threshold.
ontology_validation_report Human-readable validation summary (“Validation executed with pre-computed subclass map…” or SHACL report text). Each validation cycle.
validation_cached, validation_locked, validated_diseases_key Track whether SHACL results can be reused for a given disease list. Set during the first validation pass; reset if the disease list changes.

Conversation Enhancements

  • option_shortcuts – stores the current numbered suggestions (generic or unknown prompts). Enables replies like “option 3”.
  • acknowledged_diseases – prevents repeating the “I understand you’re working on …” acknowledgement.
  • final_purpose_payload – latest valid JSON payload ready for export/download.
  • validation_report_payload – JSON serialisation of the validation summary for download.
